# Machine Learning Intermedio

## Descripción del curso: 

Este curso proporciona una introducción práctica a MLOps (Machine Learning Operations), centrado en la integración de machine learning con DevOps. A lo largo del curso, se abordarán temáticas clave como el diseño de arquitecturas de ML confiables, la integración y entrega continua (CI/CD), y la automatización de procesos de reentrenamiento e inferencia en entornos productivos. Los participantes aprenderán a utilizar herramientas y tecnologías clave para soportar el ciclo de vida completo de proyectos de Machine Learning de manera eficiente y escalable.
